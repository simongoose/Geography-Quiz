PROJECT TITLE: Geography Quiz
PURPOSE OF PROJECT: A quiz for country to capital and for capital to country
VERSION or DATE: 09/11/2021
HOW TO START THIS PROJECT: Download .zip file and extract the folder inside. Open the folder in a java compiler (BlueJ recommended) and create a new GUI object.
AUTHORS: Simon Lee
USER INSTRUCTIONS: 
You can find this information under the information tab in game as well:
- To start the game press Play Game
- You can then select the difficulty, and the game will start
- Each game you have 3 lives, one is taken away when you get a question wrong
- You get one point for every right answer
- When you run out of lives, the game ends
- The game also ends if you have finished the difficulty (answered all questions)
- The switch mode button lets you choose between 'country to capital' and 'capital to country'
- Statistics shows your various stats and highscores
